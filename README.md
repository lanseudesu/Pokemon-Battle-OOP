# Pokémon Battle Engine

A Java-based turn-based battle engine inspired by Pokémon battles, built to practice object-oriented programming.

> **Status:** 🚧 Work in Progress

---

## Current Features

* Pokémon entities with configurable data
* Battle state management
* Move execution system
* Move PP tracking
* Accuracy checks
* Type immunity validation

    * Normal → Ghost
    * Ground → Flying
* Basic battle logging through console output

---

## Current Development Focus

The project is currently focused on establishing the foundation of the battle engine and moves with damage and not status effect moves, before implementing advanced mechanics.

Current priorities include:

* Move validation
* Move execution flow
* Accuracy and hit determination
* Damage calc
* Hit, miss and after hit effects
* Turn processing architecture

---

  * etc.
